# Automação de Testes Web com Cypress — Projeto Inicial

Este repositório contém um **projeto inicial de automação de testes web utilizando Cypress**, desenvolvido durante o módulo de **Automação Web com Cypress da Qazando Academy**.

O objetivo deste projeto é aplicar na prática os fundamentos de **automação de testes end-to-end (E2E)**, explorando organização de testes, boas práticas de automação e estruturação de código utilizando **JavaScript e Cypress**.

---

# Objetivos do projeto

- Praticar automação de testes web **E2E**
- Aplicar boas práticas na escrita de testes automatizados
- Estruturar testes de forma **organizada e escalável**
- Trabalhar com **dados dinâmicos e reutilizáveis**
- Exercitar versionamento de código utilizando **Git e GitHub**

---

# Tecnologias e ferramentas utilizadas

- JavaScript
- Cypress
- Node.js
- FakerJS
- Git
- GitHub

---

# Conceitos praticados

Durante o desenvolvimento deste projeto foram aplicados conceitos importantes de automação de testes:

## Automação de testes

- Escrita de testes **End-to-End (E2E)**
- Assertions (validações)
- Inspeção de elementos
- Uso de **hooks (`beforeEach`)**

## Estrutura e organização

- Estruturação de projeto Cypress
- Organização de testes utilizando **Page Object Pattern**
- Criação de **Custom Commands** no Cypress
- Refatoração de código para melhorar reutilização e legibilidade

## Manipulação de dados

- Uso de **fixtures**
- Geração de dados dinâmicos com **FakerJS**
- Manipulação de dados para execução de testes em massa

## Testes responsivos

- Execução de testes em **diferentes tamanhos de tela**
- Simulação de múltiplos dispositivos utilizando `cy.viewport()`
- Execução automatizada para diferentes resoluções utilizando **arrays e `forEach`**

## Boas práticas de desenvolvimento

- Uso de **imports e modularização de arquivos**
- Organização da estrutura do projeto
- Versionamento de código com Git

# Como executar o projeto

## 1. Clonar o repositório:

git clone <url-do-repositorio>

## 2. Instalar as dependências

npm install

---

## 3. Executar o Cypress

**Modo interativo:**


npx cypress open


**Modo headless:**


npx cypress run

# Sobre este projeto

Este repositório representa um **projeto inicial de automação com Cypress**, onde são aplicados os principais fundamentos da automação de testes web.

Durante o desenvolvimento foram realizadas refatorações estruturais para melhorar a organização e reutilização de código, incluindo a implementação de **Page Object Pattern**, **Custom Commands**, geração de dados dinâmicos e execução de testes responsivos.

O objetivo é consolidar conhecimentos em automação de testes e servir como base para projetos mais avançados de testes automatizados.
